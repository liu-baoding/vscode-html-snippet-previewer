# Icon Resources

This directory contains the icon files for the HTML Snippet Previewer extension.

## icon.png

A 128x128 pixel PNG icon for the extension. This icon is used in the VS Code marketplace and extension list.

## How to create the icon

1. Create a 128x128 pixel square image
2. Use a design that represents HTML preview functionality
3. Save as PNG format
4. Place in this directory as icon.png
