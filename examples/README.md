# HTML Snippet Previewer Examples

This directory contains example files to test the HTML Snippet Previewer extension.

## Python Example (python_example.py)

This file contains Python code with HTML strings embedded in it. You can select the HTML strings and preview them.

## Model File Example (model_example.mo)

This file contains a model file with HTML documentation strings. You can select the HTML documentation and preview it.

## JavaScript Example (js_example.js)

This file contains JavaScript code with HTML template literals. You can select the HTML template literals and preview them.

## How to Use

1. Open any of the example files
2. Select an HTML code snippet
3. Right-click and select "Preview HTML Snippet" or "Preview HTML Snippet to Side"
4. Or use the keyboard shortcuts: Ctrl+Shift+H (preview) or Ctrl+Shift+Alt+H (preview to side)
5. The HTML preview will open in a new panel showing the rendered HTML

## Features

- Works with any file type
- Supports inline CSS and JavaScript
- Real-time preview updates when selection changes
- Error handling for invalid HTML
- Responsive preview window
